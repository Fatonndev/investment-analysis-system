# ROI Analysis System for Pipe Rolling Plant

A PHP-based application for analyzing and forecasting return on investment (ROI) for a pipe rolling plant.

## Features

- Data collection and processing of financial indicators
- ROI, NPV, IRR calculations
- Financial forecasting
- Data visualization
- Report generation

## Requirements

- PHP 7.4+
- MySQL 5.7+
- Composer

## Installation

1. Create a MySQL database
2. Run `composer install`
3. Configure database connection in `config/database.php`
4. Run migrations to set up tables

## Technologies Used

- PHP 7.4+
- MySQL
- Chart.js for visualization
- Bootstrap for UI
- PhpSpreadsheet for Excel import/export
- TCPDF for PDF generation
