# Примеры CSV файлов для импорта данных

В этом каталоге находятся примеры CSV файлов для импорта различных типов данных в систему.

## Форматы файлов:

### 1. production_data.csv
- **Назначение**: Импорт производственных данных
- **Структура**:
  - period: Период (формат: YYYY-MM или YYYY-MM-DD)
  - product_type: Тип продукции
  - quantity: Объем производства
  - unit: Единица измерения
  - revenue: Выручка
  - variable_costs: Переменные затраты
  - fixed_costs: Постоянные затраты

### 2. costs_data.csv
- **Назначение**: Импорт данных о затратах (все типы в одном файле)
- **Структура** (все поля в одном файле, неиспользуемые поля оставляются пустыми):
  - period: Период (формат: YYYY-MM или YYYY-MM-DD)
  - cost_type: Тип затрат (raw_material, energy, logistics, labor, depreciation)
  - material_type: Тип материала (для сырья)
  - cost_per_unit: Стоимость за единицу
  - quantity_used: Количество использованного
  - total_cost: Общая стоимость
  - energy_type: Тип энергоносителя (для энергии)
  - route: Маршрут (для логистики)
  - department: Отдел (для заработной платы)
  - salary_cost: Зарплатные затраты
  - benefits: Надбавки
  - asset_name: Название актива (для амортизации)
  - depreciation_amount: Сумма амортизации

### 3. costs_raw_material.csv
- **Назначение**: Импорт данных о затратах на сырье
- **Структура**:
  - period: Период
  - cost_type: Тип затрат (должен быть 'raw_material')
  - material_type: Тип материала
  - cost_per_unit: Стоимость за единицу
  - quantity_used: Количество использованного
  - total_cost: Общая стоимость

### 4. costs_energy.csv
- **Назначение**: Импорт данных о затратах на энергоносители
- **Структура**:
  - period: Период
  - cost_type: Тип затрат (должен быть 'energy')
  - energy_type: Тип энергоносителя
  - cost_per_unit: Стоимость за единицу
  - quantity_used: Количество использованного
  - total_cost: Общая стоимость

### 5. costs_logistics.csv
- **Назначение**: Импорт данных о затратах на логистику
- **Структура**:
  - period: Период
  - cost_type: Тип затрат (должен быть 'logistics')
  - route: Маршрут
  - cost_per_unit: Стоимость за единицу
  - quantity_used: Количество использованного
  - total_cost: Общая стоимость

### 6. costs_labor.csv
- **Назначение**: Импорт данных о затратах на заработную плату
- **Структура**:
  - period: Период
  - cost_type: Тип затрат (должен быть 'labor')
  - department: Отдел
  - salary_cost: Зарплатные затраты
  - benefits: Надбавки
  - total_cost: Общая стоимость (автоматически вычисляется как salary_cost + benefits)

### 7. costs_depreciation.csv
- **Назначение**: Импорт данных об амортизации
- **Структура**:
  - period: Период
  - cost_type: Тип затрат (должен быть 'depreciation')
  - asset_name: Название актива
  - depreciation_amount: Сумма амортизации

### 8. product_prices.csv
- **Назначение**: Импорт данных о ценах на продукцию
- **Структура**:
  - period: Период
  - product_type: Тип продукции
  - size_spec: Размерная спецификация
  - precision_class: Класс точности
  - region: Регион
  - price_per_unit: Цена за единицу

### 9. investment_data.csv
- **Назначение**: Импорт данных об инвестициях
- **Структура**:
  - investment_type: Тип инвестиции
  - description: Описание
  - amount: Сумма
  - investment_date: Дата вложения (формат: YYYY-MM-DD)

## Требования к CSV файлам:
- Используйте запятую (`,`) как разделитель
- Обязательно включайте заголовки столбцов в первую строку
- Используйте формат даты YYYY-MM или YYYY-MM-DD
- Числа должны использовать точку как десятичный разделитель
